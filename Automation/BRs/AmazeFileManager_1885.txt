Bug Report Title:
 Screen rotation when AmazeFileManager requests permission will cause AmazeFileManager black screen

Bug Report Issue:
**Describe the bug**
Screen rotation when  AmazeFileManager requests permission will cause AmazeFileManager to get stuck and then black screen

**To Reproduce**
Steps to reproduce the behavior:
1. Install AmazeFileManager and open app
2. The permission request box appears
3. Rotate screen
4. Black Screen

**Expected behavior**
No black screen and can accept permissions

**Screenshots**
Black screen appears

**Smartphone (please complete the following information):**
 - Device: HUAWEI nova 5 pro
 - OS: EMUI 9.1.1
 - Rooted: No
 - Version: v3.4.3
