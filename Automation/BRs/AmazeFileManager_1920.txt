Bug Report Title:
FTP server cannot be used without WiFi, and there is no prompt

Bug Report Issue:
**Describe the bug**
When I use the data network, I can't turn on the FTP server, and there is no prompt. I can only see that the start button is gray. After trying it out, I found that this feature can only be used on WiFi networks.

**To Reproduce**
Steps to reproduce the behavior:
1. Install and open AmazeFileManager
2. Open menu bar
3. Scroll down to 'FTP server'
4. Click on 'FTP server'
5. Unable to click the start button

**Expected behavior**
Users should be prompted to turn on WiFi when there is no WiFi network, otherwise they don't know why they can't use FTP server
